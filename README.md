# Визуализация 5-мерного гиперкуба

## Описание проекта

Интерактивное приложение для визуализации 5-мерного гиперкуба в 2D-пространстве с использованием методов проекции и вращения в многомерном пространстве. Проект реализован на C++ с использованием WinAPI и GDI для графического интерфейса и отрисовки.

## Технологии и библиотеки

- **Язык программирования**: C++ (стандарт C++11)
- **Графический интерфейс**: WinAPI
- **Графическая подсистема**: GDI (Graphics Device Interface)
- **Система сборки**: CMake
- **ОС**: Windows

## Особенности реализации

### Математическая основа
- Проекция из 5-мерного пространства в 2D с использованием матрицы камеры 5×2
- Реализация матричных операций для n-мерного пространства
- Система вращения с 10 степенями свободы (C(5,2) = 10 плоскостей вращения)
- Скалярное произведение и операции с векторами различной размерности

### Алгоритм визуализации
1. Генерация вершин 5-мерного гиперкуба (2^5 = 32 вершины)
2. Соединение вершин рёбрами (вершины с расстоянием Хэмминга = 1)
3. Применение матричных преобразований для вращения в многомерном пространстве
4. Ортогональная проекция 5D → 2D с использованием камеры
5. Отрисовка спроектированных рёбер на экране

### Ключевые компоненты кода
- `Point2DByVector()` - проекция 5-мерной точки в 2D пространство
- `matrixRotation()` - создание матрицы вращения для n-мерного пространства
- `DrawHypercube()` - отрисовка 5-мерного гиперкуба
- `matricesMultiplication()` - умножение матриц произвольной размерности

## Сборка и запуск

### Требования
- Компилятор с поддержкой C++11
- CMake версии 3.10 или выше
- Windows OS

### Инструкция по сборке
```bash
# Создание директории для сборки
mkdir build
cd build

# Генерация проекта CMake
cmake ..

# Сборка проекта
cmake --build .

# Запуск приложения
./MyGraphProject
```

## Управление

### Активация управления
- Нажмите левую кнопку мыши в окне приложения для активации режима управления

### Управление вращением
- Клавиши 0-9 (основные или на NumPad): вращение вокруг различных плоскостей (каждая клавиша увеличивает соответствующий угол на 15°)

### Управление положением камеры
- Клавиши A, S, D, F, G: перемещение камеры в отрицательном направлении по осям X, Y, Z, W, V соответственно
- Клавиши Q, W, E, R, T: перемещение камеры в положительном направлении по осям X, Y, Z, W, V соответственно

## Особенности реализации

1. **Двойная буферизация**: предотвращение мерцания при отрисовке
2. **Кастомные математические операции**: реализация векторных и матричных операций
3. **Аппаратно-независимое рисование**: использование HDC для переносимости
4. **Обработка пользовательского ввода**: система управления через сообщения Windows

## Математические детали

- Размерность пространства: 5 измерений (X, Y, Z, W, V)
- Количество вершин гиперкуба: 2⁵ = 32
- Количество рёбер: 5 × 2⁴ = 80
- Количество плоскостей вращения: C(5,2) = 10
- Матрица камеры: 5×2 (проекция из 5D в 2D)
